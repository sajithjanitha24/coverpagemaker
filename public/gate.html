1. Text Input Formatting

Please follow these rules carefully for text inputs:

  • Topics: When adding multiple topics, enter each one on a separate line. Example:
    1st line = ELECTRICAL
    2nd line = Electronics
  • Main Topic: If your main topic is too long, split it into two lines. Example:
    1st line = Induction
    2nd line = Motors
  • Group Members: The group members bar will automatically place a space before the 1st member. Do not cut or remove this space.

2. Final Check

This tool is an assistant, not a replacement for your work. You are responsible for the final report. You must proofread the generated document for any errors before submitting it.
